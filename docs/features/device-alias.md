# 设备备注

> 关联决策: [ADR-0002](../adr/0002-fingerprint-as-device-identity.md)

## 概述

局域网多设备场景下,默认设备名(通常是主机名)难以辨认目标设备,容易发错对象。本功能允许给任意发现到的设备设置自定义备注,永久保留。

## 能力

- 右键设备行 → 修改备注 / 清除备注;设备名行内变输入框编辑(回车或点别处提交,Esc 取消)
- 备注完全替换默认名显示;设备列表与聊天标题栏显示一致
- 备注按设备身份永久保留:设备离线、改名、换 IP 后重新出现,备注自动恢复
- 清除备注即恢复对方的默认名
- 保存失败时就地提示且不丢输入,可重试

## 边界与不做

- 备注仅存本机,不跨设备同步
- 不做失效备注的自动清理
