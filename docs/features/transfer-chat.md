# 传输与聊天

> 关联决策: [ADR-0003](../adr/0003-chat-persistence-node-sqlite.md)

## 概述

局域网传个文件常要绕道 U 盘或网盘,一段文本更是没有轻量通道。Transfer 把文本与文件统一为聊天气泡流:选中设备即可像 IM 一样互发,记录本地保存。

## 能力

- 发送文本:即时送达并显示,对方无需确认
- 发送文件:支持一次多个(并行传输),自带完整性校验
- 接收确认:文件以气泡形式等待接收/拒绝;可开启自动接收——小于阈值的文件自动收(默认关,全部文件需确认;文本始终自动接收)
- 聊天历史本地持久化,重启后仍在
- 接收的文件与图片落系统下载目录;文件气泡可一键定位到所在文件夹;图片气泡可预览、右键用系统程序打开
- 传输进度实时显示
- 发送失败给出明确原因:对方离线提示"已离线";连不上(如对方开 VPN)约 10 秒内提示连接超时,不长时间挂起
- 聊天输入区支持直接截图与粘贴图片发送

## 边界与不做

- 不做断点续传;不做独立的传输历史页面(历史即聊天流)
- 大文件传输中断需整体重发
