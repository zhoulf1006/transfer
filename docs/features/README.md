# 功能目录(Features)

本目录描述**当前版本已具备功能的现行行为**,面向用户的产品全景。

- 不变量:出现在本目录 = 当前版本可用;行为变更就地改写;功能移除删除文件。
- 未来计划见 [DESIGN.md](../DESIGN.md) §10 迭代路线;历史见 git/Release Notes;实现与决策见 [docs/adr](../adr/)。
- 内容规则:只写用户可见行为,零实现细节;术语从 [CONTEXT.md](../../CONTEXT.md);设置项写在其所属功能内,分发渠道不入目录。

| 功能 | 一句话 |
|---|---|
| [设备发现与连接](discovery.md) | 同一局域网内自动发现设备,加密直连,免配置 |
| [传输与聊天](transfer-chat.md) | 文本与文件以聊天气泡流互发,历史本地保存 |
| [截图与标注](screenshot.md) | 全局快捷键框选截图,全套标注,一键发给聊天对象 |
| [设备备注](device-alias.md) | 给发现到的设备起自定义名字,永久保留 |
| [新消息提醒](unread.md) | 系统级角标/闪烁 + 应用内未读计数 |
| [界面与通用](general.md) | 主题、界面语言、下载目录入口 |
