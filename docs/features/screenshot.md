# 截图与标注

> 关联决策: [ADR-0008](../adr/0008-screenshot-scope.md)

## 概述

聊天中经常要"指着屏幕说话",截图-保存-找文件-拖发的路径太绕。全局快捷键一键框选截图、标注,直接发给当前聊天对象。

## 能力

- 全局快捷键唤起(默认 F1,可在设置中自定义;与系统/他人冲突时提示换键,改后即时生效)
- 区域框选,配取色放大镜精确定位
- 标注工具:矩形、椭圆、直线、箭头、画笔、文字、序号、马赛克、高斯模糊;样式(颜色/粗细)可调
- 三个出口:发送到当前聊天 / 复制到剪贴板 / 保存为文件
- 标注工具栏跟随界面语言

## 边界与不做

- macOS 首次使用需授予"屏幕录制"权限(有引导提示)
- 不做钉图、OCR、滚动长截图、录屏/GIF、延时截图
