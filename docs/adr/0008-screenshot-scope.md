# ADR-0008: 截图功能范围——三出口,对齐 Snipaste 标注,不做通用截图软件

- 状态: 已接受

## 背景

截图定位为「截图 → 直接发给对方」的轻量工具,与传输/聊天场景强绑定。需要划清与 Snipaste 类通用桌面截图软件的边界。

## 决策

- **三出口**:发送到聊天 / 复制剪贴板 / 存文件。
- **标注能力对齐 Snipaste**(全套标注工具)。
- **不做**:钉图、OCR、滚动长图、录屏。

## 后果

- 功能面收敛,截图始终服务于"发给对方"的主场景。
- 用户要通用截图能力时引导用专业工具,不背功能包袱。
