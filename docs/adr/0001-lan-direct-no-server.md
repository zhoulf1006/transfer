# ADR-0001: 局域网直连,无服务器、无账号

- 状态: 已接受

## 背景

Transfer 是跨平台(macOS/Windows)局域网工具:文件传送 + 截屏标注 + 笔记。需要决定基础架构:走云中转还是局域网直连。

## 决策

Electron + TypeScript;设备间**局域网直连**,无服务器、无账号体系。协议对齐 LocalSend(多播发现 + HTTP(S) 传输)。

## 后果

- 零云成本、零账号,隐私天然可控。
- 发现依赖局域网多播/广播,在企业网/代理环境下脆弱(后续 ADR-0005/0006 逐步补强)。
- 信任模型无 CA 可依,走自签证书 + TOFU(ADR-0004)。
